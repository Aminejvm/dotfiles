## Install Brew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Clone this Repo

```
git clone git@github.com:Aminejvm/dotfiles.git
```

## Install packages

```
brew bundle
```

## Symlink dotfiles

```
stow kitty nvim raycast-scripts tmux
```
